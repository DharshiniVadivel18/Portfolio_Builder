# Portfolio Builder - React.js

A React.js application for creating personal portfolios with local storage.

## Features

- **React.js**: Modern frontend framework
- **Local Storage**: Data persistence in browser
- **Responsive Design**: Works on all devices
- **Authentication**: Login/Signup pages
- **Dynamic Forms**: Add projects and experiences
- **Portfolio Display**: Beautiful portfolio generation

## Installation & Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open `http://localhost:3000` in your browser

## Usage

1. Visit the home page
2. Sign up or login
3. Fill out the portfolio builder form
4. View your generated portfolio
